AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that allows users to upload their resume in PDF format and compare it against a job description. The system extracts text from the resume, identifies relevant technical skills, and calculates a similarity score using TF-IDF and cosine similarity.

Features

Upload resume in PDF format
Extract text from resume
Identify predefined technical skills
Calculate job description match score
Display missing or unmatched skills

Technology Stack

Backend: Python, Flask
Machine Learning: Scikit-learn
PDF Parsing: PyPDF2
Frontend: HTML, Bootstrap

How to Run

Clone the repository
git clone https://github.com/ayandxb2k7/ai-resume-analyzer.git
Install dependencies
pip install -r requirements.txt

Run the application
python app.py

Open in browser
http://127.0.0.1:5000
